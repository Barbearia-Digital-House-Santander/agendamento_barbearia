Funcionamento de Agendamentos e Cancelamentos da Barbearia
  • AGENDAMENTO
    •
    • Para fazer a marcação on-line basta clicar na opção "AGENDE", preencher os dados e efetivar a solicitação. Você receberá um Número de Protocolo, guarde-o bem, pois em caso de Cancelamento será este código que deverá ser utilizado.
  • CANCELAMENTO
    •
    • Para fazer o cancelamento on-line basta clicar na opção "CANCELE", preencher os dados e efetivar a solicitação. O campo Número de Protocolo , deverá ser preenchido com o código recebido quando você realizou o agendamento.
  • REMARCAÇÕES
    •
    • Para fazer o reagendamento on-line, é preciso que você primeiro Cancele a sua marcação e posteriormente faça novamente o agendamento, inserindo a nova data desejada.
